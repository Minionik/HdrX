# 🧠 HeadHunter

> Fast. Clean. Brutal HTTP Security Header Scanner.

---

## ⚡ What It Does
HeadHunter scans a target URL and checks for all critical security headers.  
It supports authenticated requests and clearly shows what's **present** and what's **missing**.

---

## 🚀 Features
- 🔍 Detects all major security headers
- 🔐 Supports custom headers (Cookie, Bearer, JWT, etc.)
- 🎨 Clean CLI output (Green = Present, Red = Missing)
- ⚡ Lightweight & fast (pure bash)

---

## 🛠️ Usage

```bash
chmod +x headhunter.sh
./headhunter.sh
